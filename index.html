Web Layouts

Traditional flow of HTML elements are to be stacked on top of each other Block elements start on a new line and take up the full width available

Inline elements are those that starts on a new line only take up as much as necessary. Strong is an eg of inline elements
Span tags are also inline elements, that do not take up an width or height.

Inline-block elements are those that flow with the text and do not start on a new line. They also maintain their size and shape like block elements.

Float property can be used to move elements to the left or right on the page and allow text to wrap around it.
