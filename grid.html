Grid

Any HTML element can be a grid container. The display property of the container element is to the value of grid. The grid container contains grid items. Grid items are direct children of the grid container. This means the grid container is the parent element.

The grid container display property could be set to inline-grid for the items to flow like inline elements (lose width/height) in a grid format.
