Flex

Any HTML element can be a flex container. The display property of the container element is to the value of flex. The flex container contains flex items. Flex items are direct children of the flex container. This means the flex container is the parent element.

The flex container display property could be set to inline-flex for the items to flow like inline elements (lose width/height) in a flex format.
